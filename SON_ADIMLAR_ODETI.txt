## 🎯 SON ADIMLAR

### 1. SUNUCUYA BAĞLANMA:
```bash
# VMware vSphere Client üzerinden:
1. vSphere Client'a giriş yap
2. Sunucuya (ubuntu) sağ tıkla
3. "Open Console" seç
4. Terminal erişimi sağla
```

### 2. SSH ETKİNLEŞTİRME:
```bash
# Konsol üzerinden:
sudo systemctl enable ssh
sudo systemctl start ssh
sudo ufw allow ssh
```

### 3. PROJE KOPYALAMA:
```bash
# Yerel bilgisayardan:
scp -r bulutistan/ ubuntu@10.40.39.24:/var/www/
```

### 4. OTOMATİK DEPLOYMENT:
```bash
# Sunucuda:
cd /var/www/bulutistan
chmod +x deploy.sh
./deploy.sh
```

## 🔧 SORUN GİDERME

### Admin Paneli Açılmıyorsa:
```bash
# Hızlı çözüm:
sudo systemctl restart bulutistan
sudo systemctl restart nginx

# Log kontrolü:
sudo journalctl -u bulutistan -f
sudo tail -f /var/log/nginx/error.log
```

### Yaygın Hatalar:
- **502 Bad Gateway:** Gunicorn çalışmıyor
- **500 Error:** Django ayarları hatalı
- **404 Error:** URL yapılandırması hatalı
